# RoboNest

RoboNest is an e-commerce platform focused on electronics, robotics, and maker hardware. It also provides custom 3D printing and CAD design services for engineering and prototyping needs.

## Overview

This project is a full-stack web application designed to handle:

- Product browsing and search
- Shopping cart and wishlist
- User authentication
- Order management
- Service requests (3D printing and CAD work)

## Tech Stack

- Frontend: Next.js / React
- Styling: Tailwind CSS
- Backend: Firebase / API routes (or your chosen backend)
- Database: Firestore or equivalent
- Hosting: Vercel

## Features

- Clean and responsive UI
- Electronics product catalog
- Wishlist system
- Cart and checkout flow
- Admin-ready structure for product management
- Service pages for 3D printing and CAD requests

## Project Structure

- `/app` - Website routes
- `/components` - Reusable UI components
- `/lib` - Utilities and backend helpers
- `/public` - Static assets
- `/styles` - Global styles
